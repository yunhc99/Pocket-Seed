해당 프로그램에 구조는 Main을 기초로 실행하는 Java프로그램입니다.
해당 프로그램은 기본적으로 데이터 베이스와 연동하여 시작하는 프로그램입니다.

프로그램 JAVA단위
Main
컴파일 시작점

Login
로그인 GUI관련 클래스

Sign
회원가입 GUI관련 클래스

SubList
과목입력 GUI관련 클래스

ConditionIN
조건입력 GUI관련 클래스

Outcome
결과출력 GUI관련 클래스

Combination, Combination_Result
각각 과목들의 조합을 위한 클래스로 Combination은 필수과목과 선택과목,
Combination_Result의 경우 결과과목에 대한 조합결과를 반환

Day_dater
조건들에 대한 결과 값이 저장되어 있는 클래스

Sub_dater
과목정보를 저장할 클래스(과목명, 학점, 인원, 시간)

Result_process
선택과목과 필수과목을 합한 최종 과목에 대한 마지막 조건을 통한 선별을 위한 클래스

Result_process_select
선택과목과 필수과목에 대한 조합결과를 가지고 조건에 맞는 과목만 선별하게 하는 클래스
